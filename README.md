# RAG AI Knowledge Assistant

This project is an AI-powered knowledge assistant built using Retrieval-Augmented Generation (RAG). 
It retrieves relevant information from documents using vector search and generates intelligent responses using a language model.
The goal of this project is to learn modern AI engineering techniques such as embeddings, vector databases, and LLM integration.

## Features
- Convert documents into embeddings
- Store embeddings in vector database
- Perform semantic search
- Generate answers using an LLM
- API for question answering

## Tech Stack
- Python
- Sentence Transformers
- FAISS
- LangChain
- FastAPI
- Git

## Learning Goals
This project was built to learn:
- Retrieval Augmented Generation (RAG)
- Vector similarity search
- AI application development
- Building APIs for ML systems
